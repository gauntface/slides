Web Componentns

The look and feel is seperate from behaviour by using web components.

1. Install the web components

yarn install gf-slide-container gf-slide

Create your slide deck.

<gf-slide-container>
  <gf-slide>
    <h1>My First Slide</h1>
  </gf-slide>
</gf-slide-container>

Build Up

Adding a `build` attribute will make the slide build those elements in with each progression.
